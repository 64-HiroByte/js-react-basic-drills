# AI 学習用 解説リクエスト テンプレ集

目的：

- AI に「答え」ではなく「思考の補正」をさせる
- 自分の判断力を伸ばしながら学習効率を上げる
- React / JS / 設計レビューに強くなる

---

## 🧠 基本テンプレ（最重要）

```text
以下の問題について、自分なりに考えた結論を書きます。
その上で、模範的な思考プロセスと解説をください。

【問題】
（問題文を貼る）

【自分の解答】
（自分の判断・コード・理由）

【お願いしたい解説】
- なぜその判断になるのか
- 他の選択肢はなぜダメなのか
- 実務での判断基準
```

---

## 🔍 React / Hooks 判断特化テンプレ

```text
以下のコードについてレビューしてください。

【コード】
（コード）

【自分の判断】
- state：要 / 不要（理由）
- useEffect：要 / 不要（理由）
- useMemo / useCallback：要 / 不要（理由）

【解説してほしい点】
- 判断が正しいか
- 危険な点があれば具体的に
- より良い書き方があれば理由付きで
```

## 🧪 「消せるか？」特化テンプレ（高成長）

```text
以下のコードから「削除できるもの」があれば指摘してください。

【コード】
（コード）

【自分の予想】
- 削除できそうな state：
- 削除できそうな effect：
- 削除できそうな memo / callback：

【解説してほしいこと】
- なぜ削除できる / できないか
- 削除した場合に何が起きるか
```

以下のコードで stale closure や依存配列の問題があるか確認してください。

【コード】
（コード）

【自分の理解】
- この値は closure に閉じ込められていると思う

## 🧠 設計レビュー想定テンプレ

```text
以下はPRレビューを想定したコードです。

【コード】
（コード）

【自分のレビューコメント案】
（自分ならどう指摘するか）

【解説してほしい点】
- 指摘として適切か
- 追加で見るべき観点

```

---

## 🎯 使い方の原則（超重要）

- 必ず **自分の考えを先に書く**
- 「正解を聞く」のではなく「判断を検証する」
- 1 問 1 修正でも OK
